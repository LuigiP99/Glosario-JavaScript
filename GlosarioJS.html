Glosario de elementos del
lenguaje JavaScript

Metodo abs()

El método abs () devuelve el valor absoluto de un número.

Ver codigo aqui
function myFunction() { document.getElementById("demo").innerHTML = Math.abs(-7.25); }

Metodo alert()

Despliega un cuadro de alerta en el navegador

Ver codigo aqui
function FunctionAlert() { alert("Hola! Soy un cuadro de alerta!"); }

Propiedad de navegador appCodeName

Obtiene el nombre en clave de su navegador

Ver codigo aqui
function FuntionCodeName() { var x = "Browser CodeName: " + navigator.appCodeName; document.getElementById("demo").innerHTML = x; }

Propiedad de body

Cambia el color de fondo del actual documento

Ver codigo aqui
function FuntionBody() { document.body.style.backgroundColor = "gray"; }

Propiedad characterSet

Muestra la codificación de caracteres para este documento

Ver codigo aqui
function FunctionCharSet() { var x = document.characterSet; document.getElementById("demo3").innerHTML = x; }

Propiedad classList

Agrega la clase "mystyle" a un elemento

Click en el boton para añadir la clase "mystyle" al DIV

Soy un elemento DIV!

Ver codigo aqui
function FunctionClassList() { document.getElementById("myDIV").classList.add("mystyle"); }

Metodo click()

Simula un click del mouse al mover el puntero del mouse sobre una casilla de verificación

Pase el cursor sobre la casilla de verificación para simular un click del mouse

Ver codigo aqui

Codigo del checkbox:

<form> <input type="checkbox" id="myCheck" onmouseover="FunctionClick()" onclick="alert('click event occured')"> </form>

Codigo JavaScript:

function FunctionClick() { document.getElementById("myCheck").click(); }

Metodo confirm()

Despliega una ventana de confirmacion

Ver codigo aqui
function FunctionConfirm() { confirm("Presiona un boton!"); }

Declaracion for para JavaScript

Recorre un bloque de código cinco veces

Ver codigo aqui
function FunctionFor() { var text = ""; var i; for (i = 0; i < 5; i++) { text += "El numero es: " + i + "
"; } document.getElementById("demo4").innerHTML = text; }

Metodo getHours()

Retorna la hora, acorde a la hora local

Ver codigo aqui
function FunctionHours() { var d = new Date(); var n = d.getHours(); document.getElementById("demo5").innerHTML = n; }

Metodo log()

Retorna el logaritmo natural de un numero, en este caso el numero "2"

Ver codigo aqui
function FunctionLog() { document.getElementById("demo6").innerHTML = Math.log(2); }

Metodo max()

Retorna el numero con el mayor valor

Entre 5 y 10, ¿Que numero tiene mayor valor?

Ver codigo aqui
function FunctionMax() { document.getElementById("demo7").innerHTML = Math.max(5, 10); }

Metodo print()

Permite imprimir el documento actual

Ver codigo aqui
<button onclick="window.print()">Probar aqui</button>

Funcion String()

Convierte diferentes objetos en strings

Ver codigo aqui
function FunctionString() {
var x1 = Boolean(0);
var x2 = Boolean(1);
var x3 = new Date();
var x4 = "12345";
var x5 = 12345;

var res =
String(x1) + "
" + String(x2) + "
" + String(x3) + "
" + String(x4) + "
" + String(x5);
document.getElementById("demo8").innerHTML = res; }

Metodo sign()

Encuentra si un numero es positivo o negativo

Ver codigo aqui
function FunctionSign() {
var a = Math.sign(3);
var b = Math.sign(-3);
var c = Math.sign(0);

var x = a + "
" + b + "
" + c; document.getElementById("demo9").innerHTML = x; }
